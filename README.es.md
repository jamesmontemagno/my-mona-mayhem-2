## 🏆 Lo que Construirás

![Mona Mayhem Screenshot](https://github.com/user-attachments/assets/5eca79e2-cb9f-4e93-aa0d-23666ebde3b7)

Una **arena de batalla arcade retro** que enfrenta a dos usuarios de GitHub comparando sus gráficos de contribuciones. Ingresa cualquier par de nombres de usuario y observa el drama pixelado desarrollarse.

**Irás de esta plantilla → una app completamente funcional** usando GitHub Copilot para manejar arquitectura, implementación, estilos y pulido.

---

## 🎯 Elige tu Track

Este taller funciona en **dos tracks paralelos** — elige el que mejor se adapte a tu flujo de trabajo:

| | Track VS Code | Track CLI |
|---|---|---|
| **Ideal para** | Desarrolladores centrados en el editor | Usuarios avanzados de terminal |
| **Herramientas clave** | Chat, Plan Mode, Agent Mode, background agents | `copilot`, `/plan`, `/fleet`, `/delegate`, `/review` |
| **Entorno** | VS Code v1.107+ | Cualquier terminal |

---

## 🧠 Lo que Aprenderás

| Habilidad | Descripción |
|-----------|-------------|
| **Ingeniería de Contexto** | Enseña a Copilot sobre tu proyecto con instrucciones y restricciones precisas |
| **Desarrollo Plan-First** | Diseña la arquitectura antes de escribir una sola línea de código |
| **Flujos de Trabajo Agénticos** | Permite que Copilot realice trabajo multi-paso en archivos — contigo en el loop |
| **Iteración de Diseño** | Transforma un scaffold básico en una experiencia arcade retro pulida |
| **Flujos Paralelos** | Divide el trabajo entre agentes y sesiones para entregar más rápido |

## 🚀 Inicio Rápido

1. **Crea tu propia copia** — haz clic en **Use this template** (o haz un fork)
2. **Elige tu track:**
   - **VS Code** → clona tu repo y ábrelo en VS Code
   - **CLI** → clona tu repo, instala `copilot` y trabaja desde tu terminal
3. **Sigue la [guía del taller →](workshop/es/00-overview.md)**

> 💡 Usa el **DevContainer** incluido para un entorno completamente pre-configurado sin configuración adicional.

## ✅ Requisitos Previos

### Compartidos
- GitHub Copilot (Pro, Business o Enterprise)
- Git
- Node.js

### Track VS Code
- VS Code v1.107+
- Extensión GitHub Copilot conectada

### Track CLI
- GitHub Copilot CLI (`copilot`)
- Node.js 22+ (o instala via Homebrew / WinGet)
